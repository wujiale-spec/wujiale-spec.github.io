第二篇博客

用hugo搭建个人博客的步骤： 1.先去官网下载hugo并解压，将hugo.exe的文件目录加在path中。重启终端，运行hugo version命令行，查看hugo版本。 2.找一个目录，在终端运行hugo new site github用户名.github.io-creator。 3.进入刚才创建的creator目录，运行git init ，然后在运行git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke（下载一个默认的主题），在执行echo ‘theme = “ananke”’ » config.toml ，然后执行hugo new posts/博客名.md （创建一篇新的文章） 4.用 vs code 打开当前目录，将其变为不是草稿状态，编辑完博客之后，在终端运行hugo server -D 会有一个链接可以在本地查看自己的博客。可以在config里面更改自己博客的语言和标题。 5.新建一个终端，运行hugo,创建了一个public目录。新建一个.gitignore文件，在里头在里头写上/public/（当前仓库不要储存public），然后cd public, 在里面执行git init 之后执行git add .在执行git commit -V (提交本地仓库)。 6.在github新建一个名为github用户名.github.io的仓库，然后执行github提供的两行命令（将public上传到远程仓库），然后在github点击settings，找到github pages ，点击所存在的地址，就能查看到所写的博客。
